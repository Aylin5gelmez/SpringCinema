# Cinema System

### Simple web-based cinema management application

The application allows you to:

* downloading the data of one session
* download a list of screening
* remove movie
* create a new movie 
* modification of film data.
* delete movie


# Technologies used:

- Java 8+,
- Spring Boot 2+,
- Spring Framework 5+
- JPA (Spring Data JPA and Hibernate),
- Postgresql
- REST API
